转发 5555 端口

Tue, Jan 7, 2025

stream { upstream android_emulator { server 192.168.50.220:5555; }

server {
	listen 50000;
	proxy_pass android_emulator;
    proxy_connect_timeout 10s;
    proxy_timeout 300s;
    proxy_buffer_size 4k;  # 设置缓冲区大小，优化传输性能
}

}

调试内网不同网段的手机，试了好几次， listen 要大才可以 = =
